Банк может открывать счета клиентов трех типов: физические лица, индивидуальный предприниматель, юридическое лицо. Ведение счетов для клиентов платное и составляет 7, 13 и 17 тыс. рублей в год соответственно. Нормативы расходования ресурсов сотрудников банка для ведения одного счета для клиента заданного типа составляют: 0,5; 0,8 и 1 час соответственно. Финансовые затраты для ведения счета для клиента заданного типа составляют: 0,6; 1,2; 1,5 тыс. руб. соответственно. Фонд человеко-часов сотрудников банка, выделенный для ведения счетов составляет 400 часов, а бюджет составляет 500 тыс. руб. За пределы фонда и объема бюджета, выделенного для ведения счетов клиентов всех типов выходить нельзя. Помогите банку спланировать распределение счетов для клиентов каждого типа с целью получения максимального дохода.
